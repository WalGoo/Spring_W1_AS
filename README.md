2. 어떤 상황에 어떤 방식의 request를 써야하나요?

데이터를 가져온다 => GET
데이터를 보내서 어떤 작업을 한다 => POST
데이터를 수정을 해야겠다 => PUT
데이터를 삭제해야겠다 => DELETE
